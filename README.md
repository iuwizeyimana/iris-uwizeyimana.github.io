# Research Website

Static GitHub Pages website for a personal research homepage.

## Pages

- `index.html`: About page placeholder
- `publications.html`: Publications page placeholder
- `beyond-research.html`: Beyond Research page with photos and polished blog text
- `assets/css/main.css`: Site styling
- `assets/img/`: Website images

## Deploying on GitHub Pages

1. Create a GitHub repository named `<your-github-username>.github.io`.
2. Upload all files in this folder to the root of that repository.
3. Commit and push to the `main` branch.
4. Open `https://<your-github-username>.github.io` after GitHub Pages finishes publishing.

## Editing

- Replace placeholder text in `index.html` and `publications.html` when ready.
- Update your title, affiliation, email, CV link, Google Scholar link, or GitHub link in the header/body as needed.
- Replace images in `assets/img/` while keeping the same filenames, or update the image paths in `beyond-research.html`.
